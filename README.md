# EuroBlade

EuroBlade は MetaTrader 5 用のグリッドナンピン EA です。フィボナッチロット配分によるナンピン戦略に、トレーリングストップと MA トレンドフィルターを組み合わせてリスクを管理します。

## 主な機能

- **グリッドナンピン** — 価格が逆行するごとにフィボナッチ比率でロットを増やしながらポジションを追加（最大25段）
- **トレーリングストップ** — 指定段数以降、平均建値からの利益が一定pipsに達するとSLを自動追従
- **MA トレンドフィルター** — 現在足・上位足・下位足のMAトレンドが逆方向の場合、新規エントリーを抑制
- **ダッシュボード** — セッション状況、ポジション情報、週次損益、直近の高重要度経済指標をチャート上に表示
- **新規エントリー ON/OFF ボタン** — 指標発表前などにワンクリックで新規エントリーを停止（既存ポジションのTP決済は継続）
- **ライトモード** — チャート全体をライトテーマに自動変更（EA削除時に元の色へ復元）
- **JPY口座対応** — 口座通貨がJPYの場合は整数・円表示、それ以外は小数2桁・USD表示
- **バックテストサマリー** — テスト完了時に取引回数・合計ロット・純損益・利率・最大DDをジャーナルに出力

## ファイル構成

```
EuroBlade/
├── Experts/
│   └── gridEA.mq5              # EA本体
├── Include/
│   └── GridEA/
│       ├── GridDashboard.mqh   # ダッシュボードクラス
│       └── euroblade_icon.bmp  # チャート右下に表示されるアイコン
├── USAGE.md                    # 使い方・パラメータ説明
├── DEVELOPMENT.md              # コンパイル・デバッグ手順
├── LICENSE                     # MIT License
└── README.md                   # このファイル
```

## クイックスタート

1. `Experts/gridEA.mq5` と `Include/GridEA/` フォルダを MT5 の `MQL5/` 以下にコピー
2. MetaEditor で `gridEA.mq5` を開いて **F7** でコンパイル
3. チャートにドラッグ&ドロップ

詳しくは [USAGE.md](USAGE.md) を参照してください。
